# TechShop — Лабораторна робота №1

Тема роботи — створення HTML-макету сайту інтернет-магазину електроніки TechShop
та діаграми прецедентів. Усі сторінки зроблені чистим HTML5, без CSS і JavaScript.

## Технології

- HTML5

## Структура

- `prototype/` — сторінки сайту (головна, каталог, товар, кошик, оформлення, вхід, реєстрація, кабінет, адмінка)
- `diagrams/` — діаграма прецедентів (use-case)

## Як запустити

Відкрити файл `prototype/index.html` у браузері. Збирати нічого не треба.
